# Trading strategies overview

Этот файл — общий каталог стратегий для Lostfast.

## Назначение
Этот раздел нужен как навигационный центр по всем стратегиям в проекте.  
Он помогает быстро понять, какие методы торговли будут описаны подробно в отдельных файлах.  
Его удобно использовать как стартовую страницу документации.

## Основные стратегии
- 02. Trend following
- 03. Mean reversion
- 04. Breakout trading
- 05. Scalping
- 06. Smart Money Concept (SMC)
- 07. Support and Resistance
- 08. Pullback trading
- 09. Day trading
- 10. Swing trading
- 11. Position trading
- 12. Grid trading
- 13. Arbitrage
- 14. Volatility & Session-Based Trading

## Как пользоваться этим набором
1. Выбери стратегию, которую хочешь изучить.
2. Открой отдельный `.md` файл по этой стратегии.
3. Прочитай логику входа, выхода, фильтры и риск-менеджмент.
4. Сравни стратегию с другими в журнале сделок.
5. Сохраняй только те подходы, которые дают стабильность на дистанции.

## Принцип работы Lostfast
- Стратегии должны быть описаны формально.
- У каждой стратегии должны быть чёткие правила.
- У каждой стратегии должны быть условия отмены входа.
- У каждой стратегии должен быть риск-профиль.
- У каждой стратегии должен быть блок ошибок и типичных ловушек.

## Что будет в каждом отдельном файле
- Суть стратегии.

- Таймфреймы.

- Риск-менеджмент.
- Психологические ловушки.
- Примеры сценариев.
- Идеи для автоматизации в Lostfast.

## Важно
Не существует стратегии, которая всегда работает.  
Любая стратегия требует тестирования, статистики и дисциплины.  
Для твоего проекта особенно важно учитывать эмоции, риск и наличие чётких правил.
