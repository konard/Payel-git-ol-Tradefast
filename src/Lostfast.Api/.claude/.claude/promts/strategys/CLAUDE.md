# .claude for Trading Strategies Module

This folder contains detailed descriptions of trading strategies for the Lostfast project.

## Назначение
- Хранить формализованные описания стратегий.
- Использовать как источник знаний для AI-агентов и разработчиков.
- Обеспечивать единый язык и структуру при описании торговых подходов.

## Правила работы с этими файлами
- Каждый файл стратегии должен следовать единому шаблону (см. 02_trend_following.md как эталон).
- При добавлении новой стратегии обязательно обновляй `01_strategies_index.md`.
- Все описания должны быть максимально конкретными и пригодными для автоматизации.
- Избегай общих фраз. Фокус на правилах, фильтрах, рисках и ловушках.

## Рекомендуемый порядок изучения
Смотри `01_strategies_index.md`.

## Структура модуля
- `01_strategies_index.md` — навигация и обзор
- `02_*.md`, `03_*.md` ... — детальные описания отдельных стратегий
- `CLAUDE.md` (этот файл) — правила работы с модулем

## Важно
Эти материалы предназначены в первую очередь для разработки Lostfast (AI-помощник, backtesting, risk management, journaling). Они не являются торговыми рекомендациями.
