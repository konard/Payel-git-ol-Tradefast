# Grid Trading Strategy

## Суть стратегии
Grid trading — это размещение серии ордеров на покупку и продажу на равных интервалах (сетке) вокруг текущей цены. Стратегия зарабатывает на колебаниях цены внутри диапазона, покупая ниже и продавая выше без необходимости угадывать направление.

## Где она работает
Отлично работает во флэте и в диапазоне:
- Боковые движения на крипте
- Валютные пары в период низкой волатильности
- Некоторые акции и индексы в консолидации

Плохо работает в сильных трендах (особенно без хеджирования).

## Главная идея
Цена постоянно двигается вверх-вниз. Вместо того чтобы ловить одно большое движение, grid trader зарабатывает много маленьких сделок на каждом колебании внутри сетки.

## Почему стратегия популярна
- Хорошо автоматизируется
- Не требует точного определения тренда
- Может приносить прибыль в боковике
- Легко масштабируется

## Основные компоненты
- Верхняя и нижняя граница сетки
- Шаг сетки (расстояние между ордерами)
- Размер позиции на каждый уровень
- Правила добавления/удаления уровней
- Механизм управления убытком при выходе из диапазона

## Базовые таймфреймы
- Определение диапазона: 4H + Daily
- Исполнение: 5m – 1H

## Логика входа
Сетка строится заранее:
- Определяется диапазон (поддержка — сопротивление)
- Расставляются buy ордера ниже текущей цены
- Расставляются sell ордера выше текущей цены
- При срабатывании ордера открывается противоположный на следующем уровне

## Где стратегия ломается
- При сильном тренде без границ
- При резком выходе цены из диапазона
- При неправильном выборе шага сетки (слишком мелкий = много комиссий)
- При отсутствии механики закрытия убыточных сеток

## Типичные ошибки
- Запуск grid без чётких границ диапазона
- Слишком агрессивный размер позиции
- Отсутствие стоп-лосса на всю сетку
- Игнорирование тренда

## Риск-менеджмент
- Обязательное ограничение максимального убытка на всю сетку
- Использование "safety orders" с умом
- Регулярный пересмотр и закрытие убыточных сеток
- Учёт комиссий и funding rate (в крипте)

## Психология
Grid trading может создавать иллюзию "лёгких денег" во флэте. Главные ловушки:
- Жадность — слишком много уровней
- Нежелание закрывать убыточную сетку
- Запуск новых сеток после убытка, чтобы "отыграться"

## Что автоматизировать в Lostfast
- Автоматическое определение диапазонов
- Динамическая сетка с адаптивным шагом
- Мониторинг PnL всей сетки в реальном времени
- Автоматическое закрытие или пауза при выходе из диапазона
- Расчёт оптимального размера позиции с учётом волатильности

## Мини-шаблон правил
- Запускать grid только внутри подтверждённого диапазона
- Максимальный риск на одну сетку — X%
- Автоматическое закрытие при пробое границы
- Учёт всех комиссий в расчётах
- Не запускать новые сетки после серии убытков

## Когда использовать
Grid trading хорошо подходит для автоматизации в Lostfast на инструментах, которые часто находятся в диапазоне. Требует очень хорошего risk-менеджмента и мониторинга.
